How it Works

Twitter Sentiment Prediction is a web app that helps you understand the sentiment of the most recent tweets for a given prompt. The app uses a machine learning model called VADAR to analyze the sentiment of the tweets and categorize them as positive, neutral, or negative.

To use the app, simply enter a prompt in the search box (e.g., "COVID-19 vaccine") and click the "Predict" button. The app will fetch the most recent tweets related to the prompt using the Twitter API and analyze their sentiment using VADAR. The results will be displayed in a table with the tweet text, sentiment score, and sentiment label.

The sentiment score represents how positive or negative the tweet is, with values ranging from -1 (most negative) to +1 (most positive). The sentiment label indicates whether the tweet is positive, neutral, or negative based on the sentiment score.

The app is built using Python, Flask, Bootstrap, and Azure, which are technologies used by data engineers and developers to build and deploy web applications. However, you don't need to know anything about these technologies to use the app. Simply enter a prompt and click the button to get insights into the sentiment of recent tweets on that topic!
